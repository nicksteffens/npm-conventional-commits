# npm-conventional-commits

Starter template with npm, husky, and conventional commits.

## Use This Template

**GitHub UI:** Click "Use this template" button above

**CLI:**
```bash
gh repo create my-new-project --template nicksteffens/npm-conventional-commits --clone
cd my-new-project
npm install
```

# Project Name

## Setup

```bash
npm install
```

## Commits

This project uses [conventional commits](https://www.conventionalcommits.org/). Commit messages are validated by commitlint.

```
feat: add new feature
fix: resolve bug
docs: update readme
chore: update dependencies
```
